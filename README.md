# SOC Analyst Training Portfolio

## About Me
I am building this repository as a hands-on cybersecurity portfolio focused on developing the practical skills required for a Junior SOC Analyst role.

My training is centered on security monitoring, alert triage, log analysis, phishing investigation, SIEM fundamentals, incident documentation, and basic threat analysis. This portfolio documents my progress through structured labs, practical exercises, and analysis notes.

## Objective
The goal of this repository is to demonstrate my learning process and technical growth as I prepare for an entry-level Security Operations Center (SOC) position.

Through these labs, I aim to strengthen skills in:
- Security event analysis
- Alert triage and escalation
- Log review and investigation
- Phishing analysis
- SIEM usage and detection workflows
- Documentation and reporting
- MITRE ATT&CK mapping
- Analytical thinking in incident response scenarios

## Repository Structure
This repository is organized by topic to keep each area of study clear and easy to review.
