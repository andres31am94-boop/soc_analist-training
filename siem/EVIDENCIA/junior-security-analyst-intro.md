# Junior Security Analyst Intro - TryHackMe

## Objetivo
Entender el rol de un analista SOC junior y practicar un flujo básico de investigación y escalamiento de alertas.

## Escenario
La room introduce el trabajo diario de un SOC Level 1, incluyendo monitoreo de alertas, colaboración con otros roles del SOC y gestión básica de incidentes.

## Conceptos aprendidos
- Qué hace un analista SOC L1
- Diferencias entre Junior Analyst, Senior Analyst, SOC Engineer, SOC Manager e Incident Responder
- Flujo básico de revisión de alertas
- Escalamiento de incidentes
- Acción defensiva básica como bloqueo de IP maliciosa

## Actividad práctica
En el laboratorio se revisaron alertas en un dashboard y se identificó una IP maliciosa, luego se escaló el caso y se bloqueó la IP en el firewall.

## Habilidades practicadas
- Triage inicial
- Validación de alertas
- Escalamiento
- Respuesta básica

## Qué me dejó esta room
Esta room me ayudó a entender mejor cómo trabaja un analista SOC junior dentro de un equipo y cómo luce un flujo básico de investigación de alertas en un entorno práctico.
